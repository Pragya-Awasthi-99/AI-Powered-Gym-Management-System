# AI-Powered-Gym-Management-System

A full-stack gym management system built with modern web technologies.
The project includes authentication, role-based dashboards, attendance tracking,
and an extensible foundation for AI-driven features.

---

## Features

- User, Trainer, and Admin authentication (JWT-based)
- Role-based access control
- User dashboard with attendance statistics
- Secure backend using Prisma + MongoDB
- Modular Express.js API architecture
- Scalable frontend using React + Vite
- Clean separation of frontend and backend

---

## Tech Stack

### Frontend
- React
- Vite
- JavaScript
- CSS

### Backend
- Node.js
- Express.js
- Prisma ORM
- MongoDB
- JWT Authentication
